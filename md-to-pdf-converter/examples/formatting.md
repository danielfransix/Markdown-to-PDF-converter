# Text Formatting Examples

This document demonstrates various text formatting options available in Markdown and how they appear in PDF output.

## Basic Text Formatting

### Emphasis

- *This text is italic* (using single asterisks)
- _This text is also italic_ (using underscores)
- **This text is bold** (using double asterisks)
- __This text is also bold__ (using double underscores)
- ***This text is bold and italic*** (using triple asterisks)
- ~~This text is strikethrough~~ (using double tildes)

### Inline Code

Use `backticks` to create inline code. For example: `print("Hello, World!")` or `const x = 42;`.
